# Breaking Newz Website

A simple static news-style webpage built using only **HTML** and **CSS**. The site features headline sections, recent events, and image-based news highlights presented in a clean, organized layout.

## Features

- **Headline Section**  
  Displays key news items with short descriptions.

- **Recent Events**  
  Includes image cards for updates such as petrol and electricity price changes.

- **Clean Layout**  
  Structured for clarity and readability across screen sizes.

- **Footer Section**  
  Provides social media links for users to stay connected.
